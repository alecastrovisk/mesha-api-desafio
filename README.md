# Api Mesha

## Desafio técnico Mesha
Criando uma API com node.js para o avaliação técnica.

# Preparando o Ambiente
  ## Clonando o Repositório 
  Primeiramente é necessário fazer um clone deste repositório na sua máquina.
  Para isto, abra um terminal, navegue até alguma pasta de sua escolha e digite ```git clone "https://github.com/alecastrovisk/mesha-api-desafio.git"``` 
  
  Após isso, instale as dependências com ```npm install``` ou ```yarn install```
  É necessário que você possua o banco **PostgreSQL** na sua máquina ou um container docker rodando na **porta 5432**
  
  ## Rodando a API
  Para iniciar a API, rode o comando ```yarn dev``` e depois rode o comando das migrations ```yarn migration:run```
# Acessando a documentação
Com o ambiente instalado, você pode acessar a documentação com as rotas da API indo no seu navegador em **http://localhost:3333/docs**
